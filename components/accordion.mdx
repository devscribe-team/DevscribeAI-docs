## Overview

Accordions let you hide and show content sections, keeping your documentation clean and scannable. Users can expand only the information they need.

Use accordions for:

- FAQs and troubleshooting guides
- Optional advanced configurations
- Detailed explanations
- Step-by-step tutorials

## Accordion Attributes

<ParameterField name="title" type="string">
The heading text displayed in the accordion header.
</ParameterField>

<ParameterField name="description" type="string">
Optional subtext below the title.
</ParameterField>

<ParameterField name="defaultOpen" type="boolean" default="false">
Set to true to expand by default.
</ParameterField>

<ParameterField name="icon" type="string">
Icon name (requires showIcon).
</ParameterField>

<ParameterField name="showIcon" type="boolean" default="false">
Display the icon.
</ParameterField>

## Best Practices

<Callout type="tip">
Use clear, descriptive titles.
</Callout>

<Callout type="warning">
Don’t hide critical info in closed accordions.
</Callout>

**Do:**

- Use descriptive titles
- Group related accordions
- Keep content focused

**Don’t:**

- Over-nest accordions
- Hide critical info
- Use for navigation
